Here are some of the projects I've worked on during my studies. They cover a range of topics including web development, data analysis,networking and other projects.

Book Rental App

"This is a simple project to create a book rental application using JavaScript. The app allows users to browse books, check availability, and rent books for a certain period. This project provided valuable experience in implementing CRUD operations and local storage with JavaScript."

Network Simulation

In this project, I implemented a practical network setup using LAN cables, as taught by Bang Jay Idoan Sihotang in the Computer Network course. The project simulates a simple network connection between computers using LAN cables, helping me better understand how local area networks (LAN) function and how devices communicate through wired connections.

Web 2.0 Website

This is the final project for the Web 2 course taught by Sir Andrew Pakpahan, where the application was developed in collaboration with my colleague, Deca Caren. The project utilized the materials taught in class, including HTML, CSS, JavaScript, VueJS, PHP, and Laravel Framework versions 10/11 as the REST API back-end, with VueJS 3 as the front-end in the form of a Single Page Application (SPA). This project provided hands-on experience in building an interactive web application using modern technologies.

UI/UX DESIGN

This is the final project for the Human-Computer Interaction course taught by Sir Raymond Maulany, where the project involved creating a design using Figma. The project focused on designing a user-friendly interface with a strong emphasis on user experience and interaction.
